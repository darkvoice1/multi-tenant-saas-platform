# 安全基线检查（Phase 5）

## 口令与认证
- Bootstrap 和管理员创建用户均校验弱口令（最少 8 位，包含大小写与数字）。
- 非 `dev` 环境禁止弱 `JWT_SECRET`（长度 < 16 或默认值）。

## CORS
- 默认仅允许本地开发源：`http://localhost:5173`、`http://127.0.0.1:5173`。

## 请求限流
- 按租户限制每分钟请求数（`max_requests_per_minute`）。
- 超限返回 `429`，附 `Retry-After`。

## 参数校验
- 项目/任务/评论/附件均有长度或大小限制。
- 附件文件名做路径与控制字符清理。
