## 第一阶段：租户模型与工程初始化
### 目标
- 明确多租户边界与数据隔离策略。

### 任务清单（完成标志）
- [ ] 初始化前后端工程（Vue3 + FastAPI）
- [ ] 初始化数据库迁移框架（Alembic）
- [ ] 设计租户数据模型（tenant/org/user/project/task）
- [ ] 设计租户上下文注入中间件
- [ ] 确定隔离方式（逻辑隔离为主，预留物理隔离扩展）
- [ ] 输出系统架构图与权限边界图

### DoD
- 请求进入服务后可识别租户上下文
- 租户基础表迁移成功
- 前后端最小可运行服务可访问

## 第二阶段：认证与权限体系
### 目标
- 支持组织内角色权限控制与企业级认证流程。

### 任务清单（完成标志）
- [ ] JWT登录、刷新令牌、登出
- [ ] OAuth2/OIDC授权流程接入（第三方或Mock）
- [ ] SSO接入方案设计与对接点说明
- [ ] RBAC权限模型（管理员/经理/成员/访客）
- [ ] 权限矩阵与接口权限清单
- [ ] 接口级权限校验装饰器
- [ ] 前端路由与按钮级权限控制
- [ ] 权限回归测试

### DoD
- 无权限用户无法越权访问
- 至少20条权限测试用例通过
- OAuth2/OIDC流程可演示或有完整方案说明

- [ ] 评论@提醒与消息通知（站内信或邮件）

- [ ] 文件附件上传与权限校验
- [ ] 文件/图片处理（大小校验、缩略图或预览）
- [ ] 前端工程化：TypeScript强类型、Pinia状态管理、权限路由
- [ ] 前端构建优化（按需加载、体积分析）
